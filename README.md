# Mon projet

To-Do-App

## Description

Planifier vos tâches avec cette To-Do-List (CRUD) avec filtres (pending, completed)

## Utilisation

Entrer votre  tâche et cliquez sur l'icône "plus" pour ajouter, sur le "crayon" pour modifier et sur la "croix" pour supprimer
